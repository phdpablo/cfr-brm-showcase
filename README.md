# CFA-BRM Showcase: Interactive Summary

This repository contains an **interactive showcase** for the tutorial "[Best Practices for Your Confirmatory Factor Analysis: A JASP and lavaan Tutorial](https://doi.org/10.3758/s13428-024-02375-7)" published in *Behavior Research Methods*.

## ðŸ“– About

This showcase provides a **dynamic and visual summary** of the CFA tutorial, demonstrating best practices for conducting confirmatory factor analysis with ordinal data in applied social sciences. The example uses the WHOQOL-BREF scale with 1,047 participants, comparing three competing models: four-correlated-factor, bifactor, and second-order structures.
